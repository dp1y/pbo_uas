üíº Penjelasan Implementasi OOP

Encapsulation

Setiap class menyembunyikan internal logic:

  • User class handle login & auth
  • Kendaraan_Manager handle business logic
  • Private attributes (_attribute)
Inheritance

Kendaraan_Pribadi dan Bus mewarisi dari Kendaraan:

  • Shared: id, nopol, waktu_masuk, petugas_masuk
  • Specific: tipe (Pribadi), kategori (Bus)
  • Reusable code: inherit methods
Polymorphism

Method overriding di subclass:

  • kendaraan_masuk() berbeda logic untuk Pribadi vs Bus
  • Hitung tarif berbeda berdasarkan tipe
  • Same method, different behavior
Abstraction

Hiding complex details:

  • Kendaraan_Manager abstracts DB queries
  • User hanya perlu call simple methods
  • SQL complexity tidak terlihat di UI
